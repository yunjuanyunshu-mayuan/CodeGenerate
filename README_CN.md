##安装方法   
方法1:
~~~
1.Android studio  
 File->Settings..->Plugins-->Browse repositores..搜索GsonFormat
2.安装插件,重启android studio
~~~

方法2: 
~~~
     1.下载 CodeGenerate.jar ,
     2.Android studio  File->Settings..->Plugins -->
 install plugin from disk..导入下载的 GsonFormat.jar ,
     3重启 android studio .
~~~
